# 🍺 ReiReal

Sistema de pedidos via QR Code desenvolvido para estudo de Análise de Requisitos, Modelagem de Domínio, Programação Orientada a Objetos (POO), Banco de Dados e Desenvolvimento Back-end com Java.

---

# 📖 Sobre o Projeto

O ReiReal é um sistema que permite que clientes realizem pedidos através da leitura de um QR Code, sem a necessidade de instalar aplicativos.

O objetivo é reduzir filas, agilizar o atendimento e melhorar a experiência do cliente em estabelecimentos como adegas, bares, lanchonetes e restaurantes.

---

# 🎯 Problema

Em muitos estabelecimentos o processo de pedido depende da interação constante entre cliente, atendente e caixa, gerando:

* Filas
* Demora no atendimento
* Erros de comunicação
* Sobrecarga da equipe

---

# 💡 Solução

Disponibilizar um QR Code para que o cliente:

1. Realize um cadastro rápido
2. Acesse o catálogo de produtos
3. Monte seu pedido
4. Escolha a forma de pagamento
5. Receba seu pedido sem necessidade de aplicativo

---

# 👥 Atores do Sistema

### Cliente

* Realizar cadastro
* Visualizar catálogo
* Criar pedidos
* Escolher forma de pagamento
* Retirar pedido

### Caixa

* Localizar pedidos
* Confirmar pagamentos
* Realizar fechamento de caixa

### Atendente

* Visualizar pedidos pagos
* Separar produtos
* Entregar pedidos

### Administrador

* Gerenciar produtos
* Gerenciar estoque
* Gerenciar usuários
* Consultar relatórios
* Configurar sistema
