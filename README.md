# ReiReal

Sistema web de autoatendimento para adegas através de QR Code.

## Funcionalidades

- Cadastro rápido
- Catálogo de produtos
- Pagamento via Pix
- Pagamento no caixa
- Separação de pedidos
- Controle de estoque
- Fechamento de caixa

## Tecnologias

- Java
- Spring Boot
- JPA
- Hibernate
- H2 Database
- Git
- GitHub
