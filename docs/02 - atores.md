# ATORES DO SISTEMA

## Cliente

### Responsabilidades

* Escanear QR Code
* Realizar cadastro rápido
* Navegar pelo catálogo de produtos
* Selecionar produtos
* Gerenciar itens do pedido
* Escolher forma de pagamento
* Realizar pagamento
* Cancelar pedido antes da confirmação do pagamento
* Acompanhar status do pedido
* Retirar pedido

---

## Caixa

### Responsabilidades

* Localizar pedidos
* Consultar pedidos
* Receber pagamentos
* Confirmar pagamentos
* Alterar forma de pagamento quando necessário
* Encaminhar pedidos pagos para separação
* Consultar movimentação do caixa
* Realizar fechamento de caixa

---

## Atendente

### Responsabilidades

* Visualizar pedidos pagos
* Separar produtos
* Atualizar status dos pedidos
* Chamar cliente pelo nome e número do pedido
* Entregar pedidos
* Confirmar entrega
* Finalizar pedidos

---

## Administrador

### Responsabilidades

* Gerenciar produtos
* Gerenciar categorias
* Gerenciar estoque
* Alterar preços
* Ativar ou desativar produtos
* Consultar vendas
* Consultar relatórios
* Gerenciar usuários
* Configurar tempo limite do pagamento Pix
* Consultar fechamento de caixa
* Realizar fechamento de caixa
* Configurar parâmetros do sistema
