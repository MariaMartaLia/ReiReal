# DIAGRAMA DE DOMÍNIO

## Relacionamentos de Negócio

* Cliente realiza Pedido

* Pedido possui ItemPedido

* ItemPedido referencia Produto

* Produto pertence a Categoria

* Pedido possui Pagamento

* Usuario atende Pedido

* Usuario realiza FechamentoCaixa

* Cliente realiza Pagamento

* Categoria organiza Produto
